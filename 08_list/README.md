# 목록 태그 속성

### ul
- 순서가 없는 목록태그
- 메뉴 표현시 많이 사용
- 불릿 기호가 생김

### ol
- 순서가 있는 목록태그
- 순서 나열 시 많이 사용
- 앞에 숫자가 붙음

### dl
- 정의 목록태그
- 용어의 뜻을 나열함
- dl > dt, dd

### list-style

```css
ul {
  list-style: none;
}
```

- 리스트의 스타일을 결정하는 속성
- ul, ol 사용시 기본 설정값 none -> 불릿 기호 또는 숫자가 없어짐
- 전체 초기화 진행중 기호를 표현하면서 사용시에는 왼쪽 padding값을 부여해야 표현이 가능함
